Display choices but allows inputting any values even not in choices. Works just like suggestion.

To use in pipeline: 

editableChoice(
  name: 'PARAM1',
  choices: ['Apple', 'Grape', 'Orange'],
  // belows are optional paramters
  defaultValue: 'Grape',
  restrict: true,
  filterConfig: filterConfig(
    prefix: true,
    caseInsensitive: true,
  ),
)
